Create And Connect with EC2 Instance

  1. Login to the AWS console
  2. Go to EC2 ->Launch Ec2 instance -> Give a name to Ec2 Instance
  3. select Application and OS Images ->type of OS
  4. Select Instance type -> cpu memory
  5. Key pair
      Create a new Key Pair
    1. Give KeyPairName
    2. Key Pair Type : RSA
    3. Private Key File Format : .pem -> use with open ssh
    4. Create key Pair
  6. Allow SSH traffic from anywhere
  7. Select number of instances
  8. Launch EC2 instance
  9. Ec2 Instance is created
  10. Connect with EC2 instance with SSH
      SSH is a network protocol that provides a secure way to access a remote server. It’s widely used for managing systems and applications remotely
    1. go to the teriminal
    2. Set Permissions for Your Key File ie .PEM file chmod 600 /path/to/your-key.pem
    3. Connect Using SSH ssh -i .pem fileLocation ubuntu@ipAddress
    4. Verify the Connection :If the connection is successful, you’ll be logged into your EC2 instance. You can now manage and configure your instance as needed.
  11. update the os with sudo apt update
