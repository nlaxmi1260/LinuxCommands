Create And Connect with EC2 Instance

  1. Login to the AWS console
  2. Go to EC2 ->Launch Ec2 instance ->add .pemfile ->ok
  3. Ec2 Instance is created
  4. Connect with ec2 instance with SSH
      SSH is a network protocol that provides a secure way to access a remote server. It’s widely used for managing systems and applications remotely
    1. go to the teriminal
    2. Set Permissions for Your Key File ie .PEM file chmod 400 /path/to/your-key.pem
    3. Connect Using SSH ssh -i .pem fileLocation ubuntu@ipAddress
    4. Verify the Connection :If the connection is successful, you’ll be logged into your EC2 instance. You can now manage and configure your instance as needed.
